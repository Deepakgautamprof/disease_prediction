# 🩺 Disease Prediction from Medical Data

## 📌 Project Overview

This project is developed as part of the CodeAlpha Machine Learning Internship.

The objective is to predict whether a patient is likely to have heart disease using machine learning techniques. The model is trained on structured medical data and provides predictions based on patient health parameters.

---

## 🚀 Features

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Random Forest Classification
- Model Evaluation
- Confusion Matrix
- Classification Report
- Model Saving using Pickle
- Streamlit Web Application

---

## 📂 Dataset

Heart Disease Dataset

Dataset Source:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset
